# IronLog PWA — Workout Tracker

Personal workout tracker with Push/Pull/Legs split, recovery monitoring, and body composition logging.

## Deploy to GitHub Pages

1. Create a repo named `ironlog` at github.com/new
2. Push these files:

```bash
cd ironlog
git init
git add .
git commit -m "Initial IronLog PWA"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ironlog.git
git push -u origin main
```

3. Go to repo Settings → Pages → Source: Deploy from branch → main → / (root) → Save
4. App will be live at `https://YOUR_USERNAME.github.io/ironlog/`

## Add to iPhone Home Screen

1. Open the URL in **Safari**
2. Tap Share → "Add to Home Screen" → Add
